# webdriverIO_appium_fw

Framework based on webdriverIO + appium

## Setup framework

Preconditions: Appium server, Node.js

    npm install -g gulp

Setup:

    npm install

## Run tests:

    npm run test
    
## Parameters

    --platform=android ( 'android' by default, 'ios' is also possible )
